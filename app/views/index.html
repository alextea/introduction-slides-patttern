Introduction slides

This prototype shows different variations of the introduction slides pattern.

Light text on a dark background can cause problems for people with dyslexia and is generally harder to read. I've created a version that uses dark text on a light background, with the addition of a thick border.

You can change between the two styles using the "Version 1 / Version 2" toggle below each slide.

Guidance

Some principles/guidance around using the introduction slides pattern:

  • Slides should have headings. You may want to visually hide the headings but as a general rule it's simpler not to.
  • One thing per slide. As the slides are used to introduce a new concept or service, each slide should contain a simple chunk of information that is easy to comprehend.
  • Use larger text (24px) in slides.
  • Use "Continue" buttons to advance to the next slide.
  • Don't use form elements (except buttons) within slides. Form elements will add extra complexity and you will need to deal with error states.
