Introduction slides

This prototype shows different variations of the introduction slides pattern.

Light text on a dark background can cause problems for people with dyslexia or other cognitive impairments. I've created a version that uses dark text on a light background, with the addition of a thick border.

You can change between the two styles using the "Version 1 / Version 2" toggle below each slide.
